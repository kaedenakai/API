# ①課題番号-プロダクト名

ランダム名言ジェネレーター

## ②課題内容（どんな作品か）

- カテゴリを選択、もしくはランダムで名言を提示します。
- 名言の翻訳ができます。
- 著者名で他の名言が検索できます。
- 著者の基本情報が検索できます。（作成中）
